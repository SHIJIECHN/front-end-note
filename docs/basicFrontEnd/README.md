# 前端基础

[[toc]]

| 术语      | 解释                                                                |
| --------- | ------------------------------------------------------------------- |
| origin    | 默认远端仓库                                                        |
| master    | 默认开发分支                                                        |
| HEAD      | 当前活跃分支的游标, 如果当前处于 master 分支，那么 HEAD 指向 master |
| HEAD^     | `HEAD` 父提交                                                       |
| tracked   | `git` 已经追踪文件                                                  |
| untracked | `git` 未跟踪的文件，也就是没有提交到 `git` 过的新文件               |
| revert    | 回退                                                                |
| stage     | 暂存的更改，此时文件还没有提交到远程                                |
