## 前端基础
- ECMAScript3/5一定要学的4个部分
> * 1、函数相关的所有知识
> * 2、预编译 -> 作用域 -> 闭包
> * 3、原型 -> 原型链 -> 继承
> * 4、对象数组的操作方法

- ECMAScript6
> * 1、基本语法和新的数据类型
> * 2、面向对象
> * 3、模块化
> * 4、异步

- DOM
> * 1、对节点、属性、样式的增删改查
> * 2、事件与处理函数
> * 3、兼容性

- 函数式编程
> * 1、纯函数
> * 2、高阶函数
> * 3、惰性函数
> * 4、节流防抖
> * 5、扁平化
> * 6、柯里化、偏函数、缓存函数、结合律、函数组合、pointfree（实践后再学）

- 网络
> * 1、前端不会网络相关知识是绝对不可以的
> * 2、AJAX、大前端的Node都需要网络相关知识
